## 项目简介

ApiHub 是一个**私有化部署的 AI API 共享池**，解决"订阅太多买不起"的痛点。

### 核心创新

| 创新 | 说明 |
|------|------|
| **积分驱动** | 贡献 API Key → 获得积分 → 排队优先 |
| **私有群组** | 邀请制，只有信任的人才能加入 |
| **透明统计** | 所有人都能看到用量、贡献、排行榜 |
| **智能调度** | 轮询、权重、故障自动转移 |

---

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Rust + Axum + SeaORM |
| 前端 | React + TypeScript + Ant Design |
| 数据库 | PostgreSQL + Redis |
| 部署 | Docker + Caddy |
